IntelliJ IDEA with OpenXava

Although OpenXava comes with an Eclipse workspace ready to use, using IntelliJ instead of Eclipse is very easy.

Creating a new OpenXava project

We're going to guide you, step by step, to creating and running a new OpenXava project in IntelliJ IDEA.

Create a new IntelliJ project

In IntelliJ choose File > New > Project...:

It will show the next dialog:

Just click on Next button to show the next step:

Click on Next again to go the only relevant step, the name and location of your project:

For Project name we put Payroll, as an example, but you should write your the name of your own project instead. For the location, don't leave the default one, instead choose path of the OpenXava workspace plus the name of your project, in this case openxava-6.3/workspace/Payroll. This is very important, otherwise your project will not run afterwards.
Click on Finish, then IntelliJ will ask you:
Click on OK, but IntelliJ is still not happy enough and asks you one more thing:
Click on This Window button.
Finally, a plain Java IntelliJ project has been created. It should look like this:
