openxava / documentation / 32 bits

OpenXava Studio (openxava.exe in Windows or openxava executable in Mac/Linux) requires a 64 bits machine to work. However, you can use OpenXava framework in a 32 bits computer, even the latest version. Just download a 32 bits Java and a 32 bits Eclipse, then switch your Eclipse workspace to use the one included in OpenXava distribution.

Install Java for 32 bits

Go to Oracle JDK 8 download page, there choose a distribution with x86 in the name (or i586 in the filename), like Linux x86 Compressed Archive or Windows x86:
Download it, run it and follow all the steps to install the JDK in your computer.

Download and install Eclipse

Eclipse 2018-09 was the last with 32 bits support, so download Eclipse 2018-09 for Windows 32 or for Linux 32. Uncompress the download file and click on the eclipse executable you have inside the eclipse folder.
After starting Eclipse open the OpenXava workspace with it (File > Switch Workspace):

Follow the Getting Started guide

Now you can follow the Getting Started guided step by step without any problem, because OpenXava Studio is Eclipse-based so has the same menus and options than your Eclipse.
